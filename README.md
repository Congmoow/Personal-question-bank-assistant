# 个人题库助手

一个基于 React + Electron 的本地题库管理应用，支持多题库管理、题目分类、随机抽题、明暗主题切换等功能。

## 功能特性

### 📚 题库管理
- **多题库支持**：创建、删除、管理多个独立题库
- **题目分类**：支持单选题、多选题、填空题、判断题四种类型
- **智能导入**：支持从 JSON 文件批量导入题目
- **数据导出**：可将题库导出为 JSON 文件进行备份

### ✏️ 题目编辑
- **便捷录入**：默认提供 A、B、C、D 四个选项，可动态添加更多
- **判断题专用**：判断题提供直观的正确/错误按钮
- **编辑功能**：支持修改已录入的题目内容
- **删除功能**：可删除不需要的题目

### 🎲 测试功能
- **随机抽题**：随机抽取一道题目进行自我测试
- **答案控制**：可选择查看或隐藏答案
- **连续测试**：支持下一题继续测试

### 🎨 界面特性
- **明暗主题**：支持浅色/深色主题切换，保护视力
- **响应式设计**：适配桌面和移动设备
- **现代化UI**：基于 Tailwind CSS 的精美界面
- **本地存储**：所有数据保存在本地，无需网络连接

## 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **桌面应用**：Electron
- **图标库**：Lucide React
- **数据存储**：LocalStorage

## 使用指南

### 创建题库

1. 打开应用，在首页点击"创建新题库"卡片
2. 输入题库名称（如"期末复习"、"面试题库"等）
3. 点击"创建"按钮

### 添加题目

1. 进入题库详情页
2. 在左侧表单中选择题目类型
3. 填写题目描述和答案：
   - **单选题/多选题**：填写选项，勾选正确答案
   - **填空题**：填写答案内容
   - **判断题**：点击"正确"或"错误"按钮
4. 点击"保存题目"

### 管理题目

- **编辑题目**：点击题目卡片右上角的编辑图标
- **删除题目**：点击题目卡片右上角的删除图标
- **查看答案**：点击题目底部的"查看答案"按钮

### 随机测试

1. 在题库详情页点击右上角"随机抽题"按钮
2. 系统会随机显示一道题目
3. 思考后点击"查看答案"
4. 可点击"下一题"继续测试

### 数据备份

- **导出题库**：在首页点击"导出"按钮，选择保存位置
- **导入题库**：在首页点击"导入"按钮，选择 JSON 文件

### 主题切换

在页面右上角找到明暗切换按钮：
- 左侧：浅色主题
- 右侧：深色主题

主题设置会自动保存，下次打开应用时会记住你的选择。

## 数据存储

### 存储位置

- **浏览器版本**：存储在浏览器的 LocalStorage 中
- **桌面版本**：存储在用户数据目录
  - Windows: `%APPDATA%\个人题库助手\Local Storage\`
  - Mac: `~/Library/Application Support/个人题库助手/Local Storage/`
  - Linux: `~/.config/个人题库助手/Local Storage/`

### 数据格式

题库数据以 JSON 格式存储，结构如下：

```json
[
  {
    "id": "1640995200000",
    "title": "我的题库",
    "description": "题库描述",
    "questions": [
      {
        "id": "1640995200001",
        "category": "单选题",
        "question": "题目内容",
        "answer": "A",
        "options": [
          {"key": "A", "text": "选项A", "isCorrect": true},
          {"key": "B", "text": "选项B", "isCorrect": false}
        ]
      }
    ]
  }
]
```

## 常见问题

### Q: 如何在不同设备间同步题库？
A: 使用导出/导入功能。在源设备上导出 JSON 文件，在目标设备上导入即可。

### Q: 为什么桌面版本和浏览器版本的数据不同步？
A: 它们使用不同的存储位置。浏览器版本存储在浏览器的 LocalStorage 中，桌面版本存储在 Electron 的用户数据目录中。

### Q: 如何备份数据？
A: 定期使用导出功能将题库保存为 JSON 文件，建议保存在安全的位置。

### Q: 支持哪些操作系统？
A: 当前版本支持 Windows 系统，后续可扩展支持 Mac 和 Linux。

### Q: 如何重置所有数据？
A: 删除对应存储位置的数据文件，或在应用中逐个删除题库。
